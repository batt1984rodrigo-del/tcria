# Project snapshots

This directory stores **historical** repository structure snapshots and approved/current diffs captured before the later documentation/artifact reorganization.

Use these files to:

- review prior repository layout baselines;
- inspect the approved/current comparison that existed at the time they were captured;
- preserve historical structure evidence without cluttering the repository root.

## Important

These bundled snapshots are **not a live description of the repository after the move to `docs/project-snapshots/` and `examples/audit-artifacts/`**. Treat them as archived reference material until regenerated from the new layout.
